# wireguard-yaml

### Requirements:

- Python3
- [virtualenv](https://pypi.org/project/virtualenv/)

### Setup

- Go to folder
- `virtualenv venv`
- `pip3 install --editable .`
- `. venv/bin/activate`

- Open script, modify hardcoded API key (TODO:)

### Usage

- `wireguard`

Also check `wireguard --help` for a list of options
